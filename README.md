# 香附子杂草自建数据集

## 简介
本数据集为香附子（*Cyperus rotundus*）杂草图像数据集，用于目标检测或分割任务。每张图像对应一个同名的 TXT 标注文件，标注为 **8列数值**。

## 文件夹结构

## 标注格式（8列）
每个 TXT 文件每行代表一个目标，共8个数值，空格或逗号分隔。各列含义如下：

| 列 | 名称       | 说明                     |
|----|------------|--------------------------|
| 1  | class_id   | 类别编号（0=香附子）      |
| 2  | x_center   | 中心点 x（归一化 0~1）    |
| 3  | y_center   | 中心点 y（归一化 0~1）    |
| 4  | width      | 框宽度（归一化 0~1）      |
| 5  | height     | 框高度（归一化 0~1）      |
| 6  | confidence | 置信度（人工标注为1.0）   |
| 7  | seg_area   | 分割面积（检测框填-1）    |
| 8  | angle      | 旋转角度（水平框填0）     |
